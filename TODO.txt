# Your Basic Version Final Scope

## Project Title

**Tour and Travel Booking System**

## Main idea

Admin adds tour packages.
User views packages and books a trip.
Admin confirms the booking.
User can see booking status.

That’s it.

---

# What you will build

## 1. Admin can manage packages

Admin can:

```text
Add package
View packages
Update package
Delete package
```

Example package:

```text
Manali Trip
5 Days
₹12000
20 seats
```

---

## 2. User can view packages

User sees package cards:

```text
Goa Trip
Manali Trip
Kerala Trip
```

Each card has:

```text
Image
Title
Destination
Price
Duration
Available Seats
View Details button
```

---

## 3. User can book package

User fills simple form:

```text
Name
Email
Phone
Travel Date
Number of People
```

Then system creates booking with:

```text
Status = PENDING
```

---

## 4. Admin can confirm booking

Admin sees all bookings and clicks:

```text
Confirm
Cancel
```

Status changes:

```text
PENDING → CONFIRMED
```

or

```text
PENDING → CANCELLED
```

---

## 5. User can check booking

User enters email and sees bookings:

```text
Package Name
Travel Date
People
Total Amount
Status
```

# APIs you need only


## Booking APIs


---

